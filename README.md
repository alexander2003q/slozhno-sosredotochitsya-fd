# Сложно сосредоточиться

Учебный проект по адаптивной вёрстке с переключением цветовых схем (`light` / `dark` / `auto`).

## Технологии

- HTML5
- CSS3
- JavaScript
- Vite
- ESLint
- Prettier

## Запуск проекта

```bash
npm install
npm run dev
```

## Доступные команды

- `npm run dev` — запуск локального сервера разработки
- `npm run build` — production-сборка
- `npm run preview` — предпросмотр production-сборки
- `npm run lint` — проверка JavaScript через ESLint
- `npm run lint:fix` — автоисправление ESLint-замечаний
- `npm run format` — форматирование файлов через Prettier
- `npm run format:check` — проверка форматирования

## Структура проекта

- `index.html` — основная страница
- `styles/` — стили проекта (глобальные стили, переменные, темы, компоненты)
- `scripts/` — скрипты переключения тем
- `images/` — изображения и иконки
- `fonts/` — шрифты
